# InstaAnalyzer

A modern, privacy-focused Instagram follower analysis tool that helps you discover:

* Users who don't follow you back
* Fans (users who follow you but you don't follow back)
* Account statistics from your Instagram export

Built with pure HTML, CSS, and JavaScript — no backend, no database, and no account login required.

## Features

### Privacy First

All processing happens directly in your browser.

Your Instagram data is never uploaded, stored, or sent to any server.

### Quick Import

Upload your Instagram export ZIP file and let InstaAnalyzer automatically detect:

* `followers_1.json`
* `following.json`

### Manual Import

Prefer manual uploads?

Simply upload:

* `followers_1.json`
* `following.json`

individually and start analyzing.

### Analysis Results

View:

* Total Followers
* Total Following
* Not Following Back
* Fans

### Search & Filter

Quickly search usernames within the analysis results.

### Modern Interface

* Responsive design
* Dark mode support
* English / Indonesian language switcher
* Mobile-friendly layout

## How to Get Your Instagram Data

1. Open Instagram.
2. Go to **Settings → Accounts Center**.
3. Select **Your Information and Permissions**.
4. Choose **Download Your Information**.
5. Request a download in **JSON** format.
6. Download the ZIP file provided by Instagram.
7. Upload the ZIP directly into InstaAnalyzer.

Alternatively, extract the ZIP and manually upload:

* `followers_1.json`
* `following.json`

## Usage

### Quick Import

1. Download your Instagram export ZIP.
2. Upload the ZIP file into InstaAnalyzer.
3. Wait for processing to complete.
4. View your analysis results.

### Manual Import

1. Extract the Instagram ZIP file.

2. Locate:

   * `followers_1.json`
   * `following.json`

3. Upload both files.

4. Click **Analyze**.
